PLUGIN_SQLITEVIRTUALLIBRARIES_SETTINGS_BROWSEMENUS_ABOUT_TEXT
	EN	<b>Notes:</b><br><br>- <b>The more virtual libraries you add and the more tracks they contain, the more time LMS will need to start or to rebuild your virtual libraries.</b><br>- You <b>don't have to</b> use <i>browse menus</i>. You can just create virtual libraries / library views <b>without</b> any menus.<br>- To <b>delete</b> a virtual library and its menus just <i>select <b>Remove</b> as SQLite definition</i> and press <i>Apply</i></b>. Or you could just <b>disable</b> them and keep them for later use by unchecking the <i>Enabled</i> box.<br>- Valid <b>menu names</b> can't include special characters like <b>^{}$@<>"#%?*:/|\</b><br>- The list is first sorted by enabled/disabled, then by built-in/custom.<br>- Some <b>built-in</b> virtual library definitions and their browse menus are not user-configurable (e.g. <i>Compilations Random</i> and <i>Compilations by Genre</i>).<br><br>Read <a href="https://github.com/AF-1/lms-sqlitevirtuallibraries/wiki/SVL-vlibdef-format"><b>this wiki page</b></a> for more detailed information about the correct format of SVL virtual library definition files (incl. how to add browse menus to the home menu).<br><br>
